## Overview

ApexGrowth is a web-first funnel prototype made for teams that need an easy way to collect leads, keep them in a CRM-style interface, and see which source brought each contact in. The project is delivered as a frontend demo using HTML, CSS, and Vanilla JavaScript, so it is straightforward to review, tailor, and repurpose for marketing workflows or showcase pages.

The emphasis here is on practical lead management without extra platform overhead. You get a responsive interface, browser-side persistence, and exportable records, making it simple to inspect lead activity, follow UTM attribution, and transfer data into other tools when required.

## FAQ

**Does this require a backend?**  
No backend is included in the project profile. The demo is built around frontend logic and browser storage.

**Where does the data go?**  
Lead data is stored in browser LocalStorage unless you export it as CSV.

**Can I change the layout or form fields?**  
Yes. Since it uses standard web technologies, you can edit the funnel, dashboard, and tracking fields in the source files.

**What if it looks different on mobile?**  
The UI is responsive, so it should reflow on smaller screens. If anything seems off, review the CSS layout rules in the project files.

**How do I update the version?**  
Swap in the updated build files or refresh the repository content, then update the download link or deployment target as needed.
